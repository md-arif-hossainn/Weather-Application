# Weather-Application
It helps to track weather forecast and current conditions real-time. User can keep a tap on the weather of several location.
#Tools:
1.Fragments,</br>
2.Rest API,</br>
3. Retrofit2,</br>
4. MVVM, </br>
5. Data Binding,</br>
6. Live Data,</br>
7. Glide Library.</br>
